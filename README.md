# TaskFlow — Task Management Web App

A lightweight task manager with priorities, due dates, search, and filters. Tasks persist in your browser via `localStorage`.

## Features

- Add tasks with optional notes, priority, and due date
- Mark complete / incomplete
- Edit and delete tasks
- Filter: All, Active, Done
- Search by title or notes
- Overdue highlighting
- Clear all completed tasks

## Run locally

```bash
cd task-management-app
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```
