# TaskFlow — Task Management Web App

A lightweight task management app built with React, TypeScript, and Vite.

The app helps users organize daily work with priorities, due dates, search, filters, and local browser persistence.

## Features

- Add tasks with title, notes, priority, and due date
- Mark tasks as complete or incomplete
- Edit and delete existing tasks
- Filter tasks by all, active, and completed
- Search tasks by title or notes
- Highlight overdue tasks
- Clear completed tasks
- Save tasks in the browser using `localStorage`

## Tech Stack

- React
- TypeScript
- Vite
- HTML
- CSS
- localStorage

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
npm run preview
```

## Why I Built This

I built this project to practice React state management, TypeScript structure, UI logic, filtering, search behavior, and browser-based persistence.
